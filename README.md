PlacementBuild
Overview
BuildPlugin is an Editor Tool for Unreal Engine that provides a preview placement workflow for buildable objects.
Users can select objects from the plugin editor panel and place them into the level with real-time placement previews.

Installation
Download the plugin from Fab Marketplace.
Copy the plugin folder into your project's Plugins directory.
Open your project in Unreal Engine.
Enable the plugin in the Plugin Manager: Edit → Plugins → Installed → PlacementBuild.
Restart the editor if necessary.

Usage
Open the BuildPlugin Editor Panel.
Select a buildable object from the panel.
Place it in the viewport. A real-time preview is shown before placement.
Adjust placement as needed using the plugin’s editor options.

The plugin includes an example map inside the plugin’s Content folder demonstrating usage.
Open the map from Content/ExampleMap.umap to see the workflow.

Example Map
Path: Content/ExampleMap.umap

Demonstrates the placement system and how objects interact with the editor viewport.

Dependencies
None

Notes
Editor-only plugin
Tested on Unreal Engine 5.7 on Windows
